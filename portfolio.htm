Dwarves at Hielthe Mine

Created in Unity using C# and utilizing github for version control. Procedurally generates a terrain map and various different resource types. The player gathers resources, finds npcs and completes quests for them and builds a settlement all while surviving waves of enemies.

Github Repository

Space Time Guy

Implemented in Unity using C# and utilizing github for version control. Utilizes PCG to generate a level with a randomized theme and pattern of rooms and hallways. Various pickups and weapons spawn in each room as the player fights their way to the end room portal. Once reached the player is teleported to a different level with an incremented difficulty.

Github Repository

BattleBots

Created in NetBeans IDE using Java and Forth and utilizing gitlab for version control. Can be played as either single or multiplayer, is a team oriented turn based strategy game. Where each player controls three different tanks the scout, sniper and tank and attempts to survive the longest Utilizes a java converter for Forth based AI.
